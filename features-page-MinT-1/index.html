More In No Time

A platform to access and manage activities under IEEE Kerala Section.
The ideal destination to create your own professional space, get in touch with peers and socialise within the community, now available at your fingertips.

Features

Easy to use with embedded features to plan and organise various events. Provides insight into the workings of the Section with regular updates.
Editors for descriptions

This helps you to change font size colour and do more with the descriptions that has to be provided during the booking procedure.
The user can now add upto 5 files during the procedure of booking.

Access for changes before approval

The user can view edit or cancel the booking before the approval has be processed from requests section under the booking tab of the left menu bar.

24*7 Support

Our team would be actively responding to your queries. Feel free to reach out to our support team at support@ieee-mint.org

Dashboard Analytics

Dashboard gives an overall schedule through the calendar and latest bookings. Public events can be seen and details would be given for joining such events through the calender itself. The latest 3 bookings can be easily accessed from the dashboard for view edit and cancel options.

Service Selection

Book services are now updated with more options. You can book more than one support of the same service. Create public or private events for online meetings, entry for public events would be open to all. Multiple date booking is available as a start date and end date.

Booking time can be selected from the set of options given, no customized time slots will be provided. You can also save a booking as draft and proceed with the same details later on..
